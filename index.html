UnReel

Social accountability for healthier scrolling habits.

Project Summary

UnReel supports students who want to reduce unhealthy social media behavior without hard lockouts. The product combines real-time usage visibility and friend accountability to make behavior change practical and sustainable.

Scope

This site tracks the full project lifecycle from early needfinding through medium/high-fidelity prototyping, heuristic synthesis, final demo, and final report.

Hosting

Built as a static website designed for GitHub Pages deployment with relative links for all local assets.
